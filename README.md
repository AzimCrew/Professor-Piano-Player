Professor Piano Player

Professor Piano Player is an AutoHotkey script that automatically plays sheet music for you.
It’s mostly used for Roblox piano games, but can work anywhere you need automatic key presses.

This script was originally made by Namida Kitsune and has been modified and improved by AzimCrew.
It’s now more stable, reliable, and user-friendly.

Features

Play sheet music automatically using keystrokes

Adjustable delay between notes

Pause, stop, and start buttons

Import your own sheet music from .txt files

GUI interface for easy use

Status bar to show current action

Stable and responsive

How to Use

Install AutoHotkey
 on your PC

Download the script (ProfessorPiano.ahk)

Double-click the script to run

Use the GUI to:

Paste or import your sheet music

Set the delay between notes

Press Start, Pause, or Stop

Use the F5-F7 hotkeys to control playback (optional)

Hotkeys
Hotkey	Action
F5	Start / Play
F6	Pause / Resume
F7	Stop
PgUp	Increase delay
PgDn	Decrease delay
Delete	Exit script
Example Sheet Music
C D E F G A B
[C] [D] [E] [F] [G] [A] [B]


Use brackets to group sequences if needed.
