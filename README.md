## üõ†Ô∏è How to Use

1. Install [AutoHotkey](https://www.autohotkey.com/)  
2. Download `ProfessorPiano.ahk` or `ProfessorPiano.exe`
3. Double-click the script to run  
4. Use the GUI to:
   - Paste or import sheet music  
   - Set the delay between notes  
   - Press **Start**, **Pause**, or **Stop**  
5. Optionally, use hotkeys for quick control:  

---

## ‚å®Ô∏è Hotkeys

| Hotkey | Action |
|--------|--------|
| F5     | Start / Play |
| F6     | Pause / Resume |
| F7     | Stop |
| PgUp   | Increase delay |
| PgDn   | Decrease delay |
| Delete | Exit script |
